# vue-router-app
- 뷰 라우터 예제

## vue router란?
- 웹페이지 간의 이동 방법
- 보통 브라우저에서 웹 페이지를 요청하면 서버에서 응답을 받아 다시 사용자에게 돌려주는 시간 동안 환면 상에 깜빡거리는 현상이 나타나는데,  이런 부분을 라우팅을 처리하면 화면을 깜빡임 없이 매끄럽게 전환할 수 있으면 , 더 빠르게 화면을 조작하여 사용자 경험이 향상된다.
- 싱글 페이지 애플리케이션(SPA)에서 많이 사용
